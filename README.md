# YMB イメージビューワー

軽量・高速な画像/コミックビューア。C# + WPF製。
[Hamana](http://miyano.s53.xrea.com/) のようなキビキビ動くビューアを目指している。

## 主な機能

- **高速ページ送り**: 前後数ページを裏で先読みキャッシュ、送り時の待ちなし。
  マウスホイール/矢印キー/PageUp・Down/Spaceで送り。
- **拡大縮小・回転**: マウスカーソル位置を中心にホイールズーム。ドラッグでパン。
  0/90/180/270度回転、単一表示時は自動で最大表示になる向きへ自動回転も可能。
- **Fitモード**: 全体表示(Contain) / 幅優先 / 高さ優先 / 覆う(Cover)を明示選択。
  小さい画像や縦横比が合わない画像も画面いっぱいに拡大できる。
- **見開き(2画面)表示**: 読み進行方向は左→右/右→左を切替可能。
- **簡易フィルター**: 明るさ・コントラスト・彩度・シャープネスをスライダーで調整。
- **サムネイル/ファイルリスト**: 左端にマウスを合わせると一覧がフロート表示(ホバー展開、固定表示も可)。
- **フォルダツリー**: ドライブ/フォルダをツリーから選択(遅延読み込み)。
- **お気に入りディレクトリ**: よく見るフォルダを登録してワンクリックで開く。
- **ソート切替**: 名前(自然順)/更新日時/サイズ、昇順・降順。
- **スライドショー**: 秒数を指定して自動ページ送り。
- **先頭/末尾到達時の挙動**: ループするか、前後のフォルダ(サブフォルダを深さ優先探索)へ自動移動するか選択可能。
- **全画面表示**: F11でツールバー/サイドバー/ステータスバーを含めUIを完全に隠す。
- **前回起動時の状態を復元**: 開いていたフォルダ・ページ・各種設定を次回起動時に自動で復元。
- **本体の更新確認**: GitHub上の `version.json` と現在バージョンを比較し、新しい版があればツールバーに導線を表示。

## 使い方

- フォルダをウインドウにドラッグ&ドロップ、または `Ctrl+O` で画像フォルダを開く。
- ホイール: ページ送り(`Ctrl+ホイール`または`ホイール:ズーム`ON時はズーム)。
- 画像上で右クリック: ホイールの動作(ページ送り⇔ズーム)を切替。
- 左端にマウスを合わせるとサムネイル/フォルダ/お気に入り一覧がフロート表示。

主なキー割り当て: `→/←` ページ送り、`S` 見開き、`D` 読み進行方向、`T` 一覧固定、
`R` / `Shift+R` 回転、`+/-/0` ズーム、`P` スライドショー、`F11` 全画面、`Esc` 全画面解除。

## 開発環境のセットアップ

```
cd src/Hamana.Viewer
dotnet build
dotnet run
```

## 技術スタック

- **言語/フレームワーク**: C# / .NET 8 (`net8.0-windows`) + WPF
- **画像デコード**: `BitmapImage`(WICベース)、周辺ページを非同期先読みキャッシュ
- **フィルター処理**: `WriteableBitmap`によるCPUピクセル処理(明るさ/コントラスト/彩度/簡易アンシャープマスク)、デバウンス付き非同期適用
- **設定/データ永続化**: JSON、`%APPDATA%`配下に保存(DB不使用)
- **インストーラー**: Inno Setup(`installer/setup.iss`)
- **CI/CD**: GitHub Actions(`.github/workflows/release.yml`)、masterへのpushで
  publish→インストーラービルド→GitHub Release公開まで自動化

データ保存先: `%APPDATA%\YmbImageViewer\settings.json`

## バージョニング

`1.0.0` から開始し、パッチ番号(3桁目)が `9` に達した次のビルドでマイナー番号を繰り上げる
(例: `1.0.9` → `1.1.0`)。CI(`release.yml`)が push のたびに自動判定・自動採番する。

## ダウンロード

ビルド不要で使う場合は [Releases](https://github.com/yumebi/ymb_viewer/releases/latest) から
`YmbImageViewer-Setup-<version>.exe` をダウンロードして実行するだけでインストールできる
(self-contained、.NETランタイム別途インストール不要)。

> **注意**: このインストーラーはコード署名されていません。ダウンロード・実行時に
> Windows SmartScreenが「不明な発行元」として警告を表示する場合があります。
> 「詳細情報」→「実行」で続行できます。
